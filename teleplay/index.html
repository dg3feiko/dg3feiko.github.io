Intro

This is a pure client side Telegram Bot playground mostly for fun and proof of concepts. It runs in long polling mode and uses LocalStorage for checkpointing. Before start a test drive, you need to create your bot following the guide page to get an API Token.

Debug

Debugging logs are directed to developer console, toggle on it for inspection if needed.

Declaration

This project doesn't mean to be used in production environment and have any responsibility of privacy protection.
